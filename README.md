# Clone da Netflix

Este projeto consiste em um clone simples da interface da Netflix, desenvolvido utilizando apenas **HTML** e **CSS**.  
Ele foi criado como parte de um projeto da **DIO (Digital Innovation One)** com o objetivo de praticar fundamentos do desenvolvimento front-end.

O projeto possui caráter exclusivamente educacional, não contando com funcionalidades dinâmicas, backend ou integração com dados reais.

---

## Funcionalidades
- Interface estática inspirada na Netflix  
- Estrutura visual semelhante a uma plataforma de streaming  
- Noções básicas de responsividade com CSS  

---

## Tecnologias Utilizadas
- HTML5  
- CSS3 (Flexbox)  

---

## Como Executar
1. Clone este repositório  
2. Abra o arquivo `index.html` em um navegador de sua preferência  

Nenhuma dependência adicional é necessária.

---

## Objetivo
Este projeto tem como objetivos:
- Praticar HTML semântico  
- Aprimorar o uso de CSS para construção de layouts  
- Desenvolver atenção aos detalhes visuais de interfaces
